GLOBAL Error Pages

Custom error pages for the Webserv project. Each page provides a user-friendly interface for different HTTP error codes.

400

Bad Request

Displayed when the server cannot process the request due to malformed syntax.

View Page
403

Forbidden

Shown when the client doesn't have permission to access the requested resource.

View Page
404

Not Found

The most common error page, displayed when a requested resource cannot be found.

View Page
405

Method Not Allowed

Displayed when the HTTP method is not supported for the requested resource.

View Page
415

Unsupported Media Type

Shown when the server refuses to accept the request because the payload format is in an unsupported format.

View Page
500

Internal Server Error

Shown when the server encounters an unexpected condition.

View Page
